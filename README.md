# できた！

子ども向けのあさ・よるのしたくアプリ。カードをタップするとくるっとめくれて、ぜんぶできたらお祝い画面が出ます。

## 機能

- **☀️ あさ / 🌙 よる** の2モード切り替え
- カードをタップして完了（フリップアニメーション）
- 全タスク完了でお祝い＆コンフェッティ
- アイコンとなまえを自由にカスタマイズ
- 設定はブラウザに保存（localStorage）

## デフォルトのタスク

| あさ | よる |
|---|---|
| 👕 きがえ | 🛁 おふろ |
| 🍚 あさごはん | 👕 パジャマ |
| 🪥 はみがき | 🪥 はみがき |
| 🚽 といれ | 🚽 といれ |
| 💧 かおをあらう | 📚 えほん |
| 🎒 でかける | 💤 ねる |

## 技術

- Vanilla HTML / CSS / JS（サーバー不要）
- Vercel でホスティング
