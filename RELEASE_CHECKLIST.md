# 提出前チェックリスト

App Store へ提出（Archive → Upload）する前に毎回実施する。

---

## A. プライバシー権限の静的監査

アプリが触れる**プライバシー保護対象の機能**と、Info.plist の **UsageDescription キー**を突き合わせる。
キーが無いままアクセスすると iOS がアプリを**即・強制終了**する（＝審査でクラッシュ扱い）。

### 現在の機能と必要キーの対応表

| 機能・経路 | 必要なキー | 状態 |
|---|---|---|
| 画像選択 →「写真を撮る」（カメラ） | `NSCameraUsageDescription` | ✅ 設定済み（build 5〜） |
| 画像選択 →「フォトライブラリ」 | 不要（PHPicker＝別プロセス選択UI。ユーザーが選んだ写真のみ渡る） | — |
| 画像選択 →「ファイルを選択」 | 不要（ドキュメントピッカー） | — |
| App内課金（StoreKit） | 不要 | — |
| ハプティクス・サウンド | 不要 | — |

### 監査手順（新機能を足したら毎回）

1. **新しくアクセスする対象がないか**を確認する。特に注意：
   - カメラ / マイク（動画撮影はマイクも）→ `NSCameraUsageDescription` / `NSMicrophoneUsageDescription`
   - 写真の**保存**（ライブラリへの書き込み）→ `NSPhotoLibraryAddUsageDescription`
   - ライブラリ全体の読み取り（PHPicker以外のAPI）→ `NSPhotoLibraryUsageDescription`
   - 位置情報・連絡先・音声認識・トラッキング等 → 各 UsageDescription
   - **WKWebView 内のHTMLが起点でも同じ**（`<input type="file">`・`getUserMedia` 等はネイティブ権限を要求する）
2. プロジェクト設定の確認：
   ```
   grep INFOPLIST_KEY ios/ohayo.xcodeproj/project.pbxproj
   ```
3. **ビルド成果物**で最終確認（生成後のInfo.plistが正）：
   ```
   /usr/libexec/PlistBuddy -c 'Print' <DerivedData>/Build/Products/.../ohayo.app/Info.plist
   ```

## B. 実機での手動テスト（提出前に一巡）

iPhone と **iPad の両方**で確認する（審査は両方の実機で行われる）。

### 初回起動（アプリを削除→クリーンインストール）
- [ ] 初回コーチマーク（⚙️の吹き出し）が表示される／設定を開くと消え、再起動で出ない
- [ ] デフォルト画像（カード裏＝恐竜、クリア＝消防車）が表示される

### 基本操作
- [ ] カードをめくれる（音・ハプティクス）／全部めくるとお祝い画面→タップで戻る
- [ ] あさ⇔よる切替（未購入なら よる はペイウォール）

### 画像設定（カード裏・クリア画像 × 朝・夜タブ）
- [ ] **「フォトライブラリ」**から設定できる
- [ ] **「写真を撮る」**：初回はカメラ許可ダイアログが出る → 許可 → 撮影 → 反映される（**クラッシュしない**）
- [ ] **「ファイルを選択」**から設定できる
- [ ] カメラを**拒否**した場合も落ちない（以降は 設定→できタップ→カメラ でON可能）
- [ ] ドラッグ・ズーム調整 → セーブ → メイン画面に反映

### 課金（.storekit ローカル or Sandbox）
- [ ] ペイウォール表示（価格が出る）／✕で閉じる。表示経路は4つ：🌙よるタブ／カード（表）の🌙夜タブ／クリア画像の🌙夜タブ／カード裏の無料枠使い切り
- [ ] 購入 → 全ロック解除（よる／夜のカード編集・画像／カード裏何度でも・🔒消滅）。夜タブのペイウォールから購入した場合はそのまま夜タブに移動する
- [ ] 「購入を復元」が機能する
- [ ] 無料状態（v1.2.1〜）：あさのカード（表）編集・クリア画像変更・「カードを初期状態に戻す」ができる
- [ ] 無料状態：カード裏の変更は1回だけ→2回目はロック＋CTA

### その他
- [ ] 「カードを初期状態に戻す」→ 確認ポップアップ → 全初期化（購入状態は維持）
- [ ] 設定フッター：プライバシーポリシー／利用規約 → Safariで開く、メール → メールアプリが開く
- [ ] iPad：レイアウト崩れなし（横向きも）
- [ ] 縦横回転・バックグラウンド復帰で異常なし

### 提出直前
- [ ] バージョン／ビルド番号を更新したか
- [ ] Archive は Release 構成（`kDebugForcePremium` は自動無効）
- [ ] アップロード後：ASCで輸出コンプライアンス回答（標準の暗号化のみ＝免除）
