## 概要（Description・最大4000字）

```
「できた！」は、子どもが朝と夜のしたくを自分で楽しく進められる、シンプルな支度ボードです。

「つぎは何をするんだっけ？」を、かわいいカードで見える化。できたらカードをタップするだけ。ぜんぶできたら、紙吹雪でおおよろこびのおいわいが待っています。

■ 無料で使えること
・「あさ」のしたくカード（タップでめくって達成、おいわい演出つき）
・カードをめくると1枚の絵が完成する「カードのうら」（初回の画像変更は無料）

■ プレミアム（買い切り・1回のお支払いでずっと）
・カードのアイコン・名前を自由に変更
・クリア画像／カードの裏画像を好きな写真に設定（「あさ」「よる」別々に）
・「よる」モードもぜんぶ使える
・今後追加される新機能も対象

■ こんな方に
・朝の「はやくして！」を減らしたいおうちの方
・子どもに自分でしたくをする習慣をつけたい
・やることをシンプルに見える化したい

■ 安心のプライバシー
・煩わしい広告は一切なし
・個人情報やデータの収集なし。写真は端末内にのみ保存され、外部に送信されません

毎日のしたくが、親子のちょっとうれしい時間になりますように。
```

## App Privacy（データ収集）

- **Data Not Collected**（データを収集していません）を選択。
- トラッキングなし・第三者SDKなし（アプリ版はアナリティクス無効）。

## 年齢区分アンケート

- すべて「なし / None」→ **4+** になります。

## 輸出コンプライアンス

- 暗号化の使用：標準的なHTTPS以外の独自暗号化なし → アップロードのたびに **「非対象（Exempt）」で手動回答**。
  - ※ `ITSAppUsesNonExemptEncryption` は Info.plist にハードコードしない方針（将来別の暗号化を足したときに見落とすため）。

## スクリーンショット（6.9インチ / 1320×2868）

`marketing/screenshots/` に5枚：
1. 広告なし！＋カードのうら（恐竜モザイク）　2. 好きな画像を使える！　3. 子どものやる気アップ（クリア画像・消防車）　4. おやすみの準備も（よる）　5. わが家のしたくにカスタマイズ（設定）

App Store Connect の「6.9インチディスプレイ」枠にアップロード（最大10枚）。

再生成は `python3 marketing/generate-screenshots.py` で `/tmp` にHTMLを書き出し、
Microsoft Edge ヘッドレスでPNG化（シミュレータのWKWebViewは絵文字が豆腐になるため）。
実写は使えないため、1枚目の「できたカードのうら」は **SVGで手描きしたオリジナルのゆるキャラ**
（子どものクレヨン画風・実在IPなし）を、5枚目のクリア画像は CSS の夕焼け風景を、
それぞれ「ここに自分の画像が入る」プレースホルダとして使用。
